# POZI

한국어 포즈 가이드 모바일 웹앱. React 19 + TypeScript + Vite + Express + Gemini API(향후 OpenAI).

## 하네스: POZI 개발팀

**목표:** 화면/카메라 UI, 서버 AI 통합, 포즈 콘텐츠 큐레이션, 통합 정합성 검증을 4인 에이전트 팀으로 자동 분담.

**트리거:** 다음 요청 시 `pozi-orchestrator` 스킬을 사용하라.
- "POZI에 [기능] 추가/수정", "새 화면 추가"
- "포즈 카테고리/포즈 데이터 추가/큐레이션"
- "Gemini → OpenAI 마이그레이션", "AI 프롬프트 개선"
- "UI 폴리시/애니메이션 개선"
- "검증/QA/타입체크"
- 후속: "다시 실행", "재실행", "수정", "[부분]만 다시"

단순 파일 읽기/질문은 직접 응답 가능 (오케스트레이터 거치지 않음).

**AI Provider 정책:** 사용자는 OpenAI API 키 사용 예정. 새 AI 작업은 OpenAI 기준으로 설계. 기존 `@google/genai` 코드는 사용자 명시 요청 시에만 교체.
